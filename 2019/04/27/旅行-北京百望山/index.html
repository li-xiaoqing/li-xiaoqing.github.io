旅行-北京百望山

发表于 2019-04-27 | 分类于 生活 | |

百望山简介

百望山是太行山延伸到华北平原东部山脉，素有“太行前哨第一峰”之称。每逢秋天，这里的红叶非常好看，红的醉人，是欣赏红叶的好地方。它主峰海拔210米，峰顶可以远眺京城，尤其是海淀区附近的面貌，看到高楼林立，软件园的各大互联网公司的办公区，这里是登高望远的好地方。公园里植被覆盖率较高95%以上，空气非常好，有北京城市氧源之称。公园内建有首都绿色文化碑林，有很多的碑亭、碑廊及碑刻，有毛泽东的题词、其它社会名人、书法爱好者的作品。这里祖传是杨六郎抗击辽兵的战场，佘太君为给儿助威，站到山角上眺望六郎，因此也成为望儿山。同时山上还保留着佘太君庙，山顶为望京楼，后期楼旁的路边新建了10米左右的玻璃栈道，供游客体验。

行前准备

看自己旅游的习惯，简单的了解下百望山的旅游攻略，做到心中有数。百望山其实不大，小半天就能完成整个行程，因此出发前带适量的吃的和喝的就可以了，特殊情况下山上也有几个小超市也会提供一些食品和日常用品。最后别忘记了涂防晒或者防蚊虫。

交通和费用

可以选择公交或者自驾。百望山森林公园公交站下车，就能看到公园东大门，进门左转就是售票处，成人票6元，儿童半价票3元。门口右侧是停车场，自动取卡停车计费，应该是15分钟0.5元。停车场车位不是很多，不过来百望山游玩的人不是很多，车位一般够用，百望山北门那边也有一些停车位，这次没走北门，对北门不太了解。

美食

景区附近是没有吃饭的地方的，建议吃完饭来游玩，游玩结束再美餐一顿！

购物

小吃、水果、美食和纪念品这里都没有，好好欣赏美景吧！

注意事项

山虽然比较好爬，但还是注意山路，穿比较舒适的运动鞋，带小孩的，一定要照顾好小孩，注意拍照地点及安全。

路线

上山的路有两条，一条是大路，一条小路，其实都比较好走。小路在进入大门有手边，路标是很清晰的，顺着路标走不迷路=——=。俗话说上山容易下山难，建议上山的时候从右侧的小路开始爬，返下山反程选择大路。
